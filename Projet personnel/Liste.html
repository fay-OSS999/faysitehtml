Exemples de Types de Listes en HTML

1. Liste non ordonnée (unordered list)

La liste non ordonnée utilise des puces pour les éléments :

  • Pommes
  • Oranges
  • Bananes

2. Liste ordonnée (ordered list)

La liste ordonnée utilise des numéros pour les éléments :

  1. Faire les courses
  2. Cuisiner le repas
  3. Dîner en famille

3. Liste de définition (definition list)

La liste de définition est utilisée pour les termes et leurs définitions :

HTML
Le langage de balisage pour créer des pages web.
CSS
Le langage utilisé pour styliser les pages web.
JavaScript
Un langage de programmation pour rendre les pages interactives.

4. Liste imbriquée

Il est possible d'imbriquer des listes non ordonnées ou ordonnées :

  • Fruits
    • Pomme
    • Banane
  • Légumes
    • Carotte
    • Épinards

5. Liste avec des puces personnalisées

Vous pouvez personnaliser les puces d'une liste avec CSS :

  • Chocolat
  • Vanille
  • Fraise

6. Liste avec des numéros personnalisés

Il est également possible de personnaliser les numéros d'une liste ordonnée :

  1. Première étape
  2. Deuxième étape
  3. Troisième étape
